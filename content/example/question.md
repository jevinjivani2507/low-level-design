# Parking Lot System

Design a parking lot system that supports the following:

## Requirements

- The parking lot has multiple floors, each floor has multiple parking spots.
- The parking lot can park motorcycles, cars, and buses.
- The parking lot has motorcycle spots, compact spots, and large spots.
- A motorcycle can park in any spot.
- A car can park in a compact or large spot.
- A bus can park in five consecutive large spots.
- The system should be able to track which spots are available.

## Constraints

- Assume the parking lot has 5 floors and each floor has 20 spots.
- There are 4 motorcycle spots, 12 compact spots, and 4 large spots per floor.

## Expected

- Class diagram
- Key methods and their implementations
- How you would handle concurrency
